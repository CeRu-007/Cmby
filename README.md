# Cmby

基于 Flutter 的跨平台 Emby 媒体客户端。

## 功能特性

- **跨平台支持** — Android、Android TV、iOS、Windows、macOS、Linux
- **自动弹幕匹配** — 无需服务端插件，客户端自动匹配 B站、爱奇艺、腾讯等 13 个弹幕源
- **完整的 Emby 体验** — 媒体库浏览、播放、搜索、收藏、续播等
- **首页大屏轮播** — 推荐内容的视觉化展示

## 下载

从 [Releases](https://github.com/CeRu-007/Cmby/releases) 下载对应平台的构建产物。

### iOS 说明

当前构建的 `.ipa` 为**未签名**版本，无法直接通过系统安装。请使用以下方式之一侧载安装：

- [AltStore](https://altstore.io)
- [SideStore](https://sidestore.io)
- [TrollStore](https://trollstore.app)
- Sideloadly 等工具

如果您有 Apple Developer 账号，可自行使用开发证书签名后安装。

### 平台适配说明

> Android TV、macOS 等平台因缺乏实际设备测试，界面和交互可能未完全适配，如有问题欢迎反馈。
